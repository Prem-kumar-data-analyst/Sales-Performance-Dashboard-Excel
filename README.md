# 📊 Sales Performance Dashboard in Excel

## Overview
This project showcases an interactive Sales Performance Dashboard built entirely in Microsoft Excel.

The dashboard was created using:
- Data Cleaning
- Pivot Tables
- Pivot Charts
- VBA Macros
- Excel Dashboard Design
- AI-assisted workflow (Claude AI)

The objective was to transform raw sales data into actionable business insights through interactive visualizations and KPI tracking.

## Key Metrics

## Features

### KPI Cards
- Total Sales
- Total Profit
- Total Orders
- Quantity Sold
- Average Discount
- Profit Margin

### Visual Analysis
- Sales by Segment
- Sales & Profit Trend
- Monthly Sales Performance
- Profit vs Discount Analysis
- Sales by Category
- Sales by Region

### Data Preparation
- Data Cleaning
- Missing Value Handling
- Data Formatting
- Pivot Table Creation
- Dashboard Automation using VBA

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- VBA Macros
- Claude AI

---

## Business Insights

- Consumer Segment contributes highest sales.
- Technology category generates maximum revenue.
- East region shows highest sales performance.
- Monthly trend analysis helps identify seasonal demand patterns.
- Discount impact can be evaluated against profitability.
